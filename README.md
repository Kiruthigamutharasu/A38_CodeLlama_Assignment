# CodeLlama AI Coding Assistant

## Features
- Generate Python Code
- Explain Code
- Debug Errors
- Optimize Code

## Tech Stack
- Python
- Ollama (Local LLM)
- CodeLlama Compatible
- Streamlit

## Setup Instructions

1. Create virtual environment
python -m venv venv
venv\Scripts\activate

2. Install dependencies
pip install -r requirements.txt

3. Start Ollama
ollama serve

4. Run Application
python -m streamlit run app.py
